# 🛒 Mi E-Commerce - Proyecto Final

Este es el proyecto final para mi curso de desarrollo web. Se trata de una tienda virtual funcional construida con **React**, diseñada bajo el concepto de **Atomic Design** y utilizando **Tailwind CSS** para un diseño moderno y profesional.

## 🚀 Funcionalidades Principales

- **Catálogo de Productos**: Visualización de productos con imágenes y descripciones.
- **Filtros Dinámicos**: Búsqueda por nombre y filtrado por categorías.
- **Detalle de Producto**: Página individual para cada artículo con información ampliada.
- **Carrito de Compras**: Gestión de estado global (Zustand) para añadir, eliminar y vaciar el carrito.
- **Proceso de Checkout**: Formulario de simulación de pago y envío.
- **Navegación Fluida**: Implementación de React Router para navegar sin recargar la página.

## 🛠️ Tecnologías Utilizadas

- **React** (Vite)
- **Tailwind CSS** (Estilos y diseño)
- **Zustand** (Gestión de estado global)
- **React Router Dom** (Navegación)
- **JavaScript ES6+**

## 📦 Instalación y Ejecución

Para ejecutar este proyecto en tu computadora local, sigue estos pasos:

1. **Clonar o descargar** los archivos del proyecto.
2. Abrir una terminal en la carpeta raíz del proyecto.
3. Instalar las dependencias necesarias:
   ```bash
   npm install
   ```
4. Iniciar el servidor de desarrollo:
   ```bash
   npm run dev
   ```
5. Abrir el navegador en la dirección que indique la terminal (normalmente `http://localhost:5173`).
